# Case G — So sánh Trước / Sau

## Workflow truyền thống (PowerPoint)

### Timeline
| Bước | Thời gian |
|---|---|
| Outline nội dung | 30 phút |
| Chọn template PPT | 30 phút |
| Design slide layout | 2-3 giờ |
| Viết content | 2-3 giờ |
| Format code blocks | 1 giờ (screenshot → paste) |
| Review & chỉnh | 1-2 giờ |
| Export PDF để share | 10 phút |
| **Tổng** | **7-10 giờ** |

### Hạn chế
- ❌ Code blocks không syntax highlight
- ❌ Tables khó format đẹp
- ❌ Không interactive (static slides)
- ❌ File nặng (10-50MB với images)
- ❌ Chỉnh nội dung = phải mở PPT
- ❌ Không responsive — xem trên phone rất tệ

## Workflow mới (AI Website)

### Timeline
| Bước | Thời gian |
|---|---|
| Outline nội dung | 15 phút |
| AI sinh website skeleton | 30 phút |
| AI sinh content từng section | 1-2 giờ |
| Review & iterate | 1-2 giờ |
| Tạo demo files | 30 phút |
| **Tổng** | **3-5 giờ** |

### Ưu điểm
- ✅ Code blocks: syntax highlight tự động
- ✅ Tables: render đẹp từ markdown
- ✅ Interactive: sidebar navigation, demo viewer, progress bar
- ✅ Dark/Light mode: toggle 1 click
- ✅ Folder size nhẹ (~500KB không images)
- ✅ Chỉnh nội dung = edit text file → refresh
- ✅ Responsive: mobile friendly
- ✅ Version control: git diff xem changes

## So sánh trực quan

| Feature | PowerPoint | AI Website |
|---|---|---|
| **Thời gian** | 7-10 giờ | 3-5 giờ |
| **Code blocks** | Screenshot (mờ) | Syntax highlight |
| **Tables** | Manual layout | Markdown → auto |
| **Navigation** | Prev/Next only | Sidebar + keyboard + progress |
| **Dark mode** | ❌ | ✅ |
| **Demo files** | Attach riêng | Viewer tích hợp |
| **Mobile** | ❌ Fixed size | ✅ Responsive |
| **Edit** | Mở PPT app | Edit text → refresh |
| **Share** | Email PPT (heavy) | Link hoặc folder (light) |
| **Collab** | OneDrive/SharePoint | Git + PR review |

---

## Kết quả thực tế — Website này

- **11 sections** với sidebar navigation
- **30+ demo files** (markdown) mở trong modal viewer
- **Dark/Light mode** toggle
- **Keyboard shortcuts** (← → navigate, Escape close)
- **Progress bar** hiện % hoàn thành
- **Mobile responsive** — hamburger menu
- **Toàn bộ tạo bằng AI** — từ skeleton đến content
- **Thời gian tổng:** ~5 giờ (bao gồm iterate nội dung)

> *"Website bạn đang xem chính là bằng chứng — AI có thể tạo ra sản phẩm hoàn chỉnh từ outline text."*

## Bối cảnh
Cần tạo slide sharing kiến thức cho team BrSE/Comtor về "Ứng dụng AI thực chiến". PowerPoint mất 4-8 giờ design. Quyết định thử AI tạo website presentation.

---

## Outline viết tay (15 phút)

```
Chủ đề: Ứng dụng AI thực chiến cho BrSE・Comtor
Đối tượng: Team BrSE/Comtor (12 người)
Thời lượng: ~2 giờ

Sections:
1. Tổng quan — Giới thiệu + Agenda
2. Opening — Survey results (khó khăn thực tế)
3. Bối cảnh — Tại sao AI là must-have cho BrSE
4. Bảo mật — Nguyên tắc vàng: dùng AI mà không vi phạm NDA
5. Case Study 1 — Giao tiếp tiếng Nhật (email, meeting, glossary)
6. Case Study 2 — Đọc/viết spec (summarize, Q&A, estimation)
7. Case Study 3 — Quản lý dự án (report, onboarding, tracking)
8. Case Study 4 — Ứng dụng sáng tạo khác (UI demo, tool cá nhân)
9. Prompt technique — Framework CRAFT
10. Hậu kiểm — AI output ≠ Final output
11. Closing — Lộ trình + tài nguyên

Mỗi section cần:
- Câu hỏi mở đầu (engage audience)
- Dữ liệu survey
- Khó khăn thực tế (quote)
- Giải pháp + demo
- Takeaway

Yêu cầu kỹ thuật:
- Website HTML responsive
- Sidebar navigation (click chuyển section)
- Dark/Light mode
- Demo file viewer (click mở markdown files)
- Mobile friendly
- Phím tắt ← → để navigate
```

---

## Tại sao chọn website thay vì PowerPoint?

| Lý do | Chi tiết |
|---|---|
| **Interactive** | Click navigate, demo files, code blocks |
| **Code blocks** | Syntax highlight — PowerPoint không format được |
| **Dark mode** | Audience prefer tuỳ |
| **Share dễ** | Gửi folder hoặc deploy link |
| **Version control** | Git track changes |
| **Responsive** | Xem trên phone được |

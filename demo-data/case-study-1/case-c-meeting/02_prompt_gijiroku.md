# Case C — Prompt tạo 議事録 từ transcript

```
[CONTEXT] 
Meeting giữa team offshore VN và khách hàng JP.
Dự án: Hệ thống quản lý nông nghiệp.
Cuộc họp review tiến độ Sprint 5.

[ACTION] 
Từ transcript bên dưới, viết 議事録 (meeting minutes).

[FORMAT]
1. 会議情報: 日時 / 場所 / 参加者
2. 議題一覧
3. 各議題の内容:
   - 議論内容（簡潔に）
   - 決定事項（明確に）
   - 未決事項（あれば）
4. 次回アクション: 担当者 / 内容 / 期限

[TONE] 
Formal, ngắn gọn, dùng ですます体 nhất quán.
Chỉ ghi nội dung quan trọng, không ghi câu xã giao.

---

[TRANSCRIPT - paste nội dung từ file 01_meeting_transcript.md]
```

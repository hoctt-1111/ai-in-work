# ユーザー管理画面 詳細設計
# User Management Screen — Detailed Design

**画面ID:** SCR-USR-002
**関連機能:** USR-001, USR-002, USR-003

---

## 1. 概要
システム管理者がユーザーの登録・編集・無効化を行う画面。

## 2. 画面設計

### 2.1 ユーザー一覧画面

**検索条件:** 氏名、メールアドレス、ロール、ステータス

**一覧表示項目:**
| 列 | 内容 |
| --- | --- |
| 氏名 | ユーザーの氏名 |
| メールアドレス | ログインID |
| ロール | 管理者/倉庫担当/圃場管理/経営管理/一般 |
| 所属部署 | 部署名 |
| 電話番号 | **必須** |
| ステータス | 有効/無効/仮登録 |
| 最終ログイン | 日時 |

### 2.2 ユーザー編集画面

**編集可能項目:**
| No. | 項目名 | 編集可否 | 備考 |
| --- | --- | --- | --- |
| 1 | 氏名 | ○ | |
| 2 | メールアドレス | × | 変更不可 |
| 3 | ロール | ○ | **システム管理者のみ変更可** |
| 4 | 所属部署 | ○ | |
| 5 | 電話番号 | ○ | **任意** |
| 6 | ステータス | ○ | 管理者のみ |

## 3. 権限マトリクス

| 操作 | システム管理者 | 経営管理者 | 倉庫担当 | 圃場管理者 |
| --- | --- | --- | --- | --- |
| ユーザー一覧閲覧 | ○ | ○ | × | × |
| ユーザー登録 | ○ | × | × | × |
| ユーザー編集 | ○ | ○ | × | × |
| ユーザー無効化 | ○ | × | × | × |
| ロール変更 | ○ | × | × | × |

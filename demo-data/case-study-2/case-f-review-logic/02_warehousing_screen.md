# 入庫管理画面 詳細設計
# Warehousing Screen — Detailed Design

**画面ID:** SCR-INV-004
**関連機能:** INV-002

---

## 1. 概要
倉庫担当者が、収穫された農産物を倉庫に入庫する操作を行う画面。

## 2. 画面設計

### 2.1 入庫登録画面

**入力項目:**
| No. | 項目名 | 型 | 必須 | 備考 |
| --- | --- | --- | --- | --- |
| 1 | 入庫日時 | 日時 | ○ | デフォルト: 現在 |
| 2 | 倉庫 | プルダウン | ○ | 全倉庫から選択可 |
| 3 | 品目 | プルダウン | ○ | |
| 4 | 等級 | プルダウン | ○ | |
| 5 | 入庫数量 | 数値 | ○ | 0.1〜99999.9 |
| 6 | 賞味期限 | 日付 | ○ | 自動計算 |
| 7 | 収穫ロットNo. | テキスト | **任意** | |
| 8 | 備考 | テキストエリア | - | |

### 2.2 倉庫容量チェック
- 容量超過 → エラー、登録不可
- 容量80%以上 → 警告、登録可能

## 3. 権限
- **倉庫担当者、圃場管理者、システム管理者** が入庫操作可能
- 経営管理者は閲覧のみ

## 4. ビジネスルール

### BR-INV-001: 入庫数量上限
- 1回の入庫数量は **99999.9kg** まで

### BR-INV-002: 倉庫選択
- ユーザーは全倉庫から選択可能（担当倉庫制限なし）

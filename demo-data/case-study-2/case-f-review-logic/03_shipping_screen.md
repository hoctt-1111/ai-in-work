# 出庫管理画面 詳細設計
# Shipping Screen — Detailed Design

**画面ID:** SCR-INV-005
**関連機能:** INV-004

---

## 1. 概要
出荷管理者が倉庫から出荷先への出庫を記録する画面。

## 2. 画面設計

### 2.1 出庫登録画面

**対象選択:**
- 在庫一覧から出庫対象を選択（複数選択可）
- FIFO原則: 賞味期限が近いものを優先表示

**入力項目:**
| No. | 項目名 | 型 | 必須 | 備考 |
| --- | --- | --- | --- | --- |
| 1 | 出庫日時 | 日時 | ○ | |
| 2 | 出荷先 | プルダウン | ○ | |
| 3 | 出庫数量 | 数値 | ○ | 在庫数量以下 |
| 4 | 出荷伝票No. | テキスト | ○ | |
| 5 | 備考 | テキストエリア | - | |

### 2.2 出荷先制約
- **JAグループ → 等級「秀」「優」「良」すべて出荷可能**
- 直売所 → 等級制限なし
- 加工業者 → 等級「良」「外」のみ

## 3. 権限
- **出荷管理者のみ** が出庫操作可能
- 倉庫担当者は出庫操作**不可**
- システム管理者は全操作可能

## 4. ビジネスルール

### BR-SHP-001: マイナス在庫防止
- 出庫数量が在庫数量を超える場合 → エラー

### BR-SHP-002: 廃棄済み在庫
- ステータスが「要廃棄」の在庫も出庫可能（値引き出荷用）
- ただし、賞味期限切れの在庫は出庫不可

### BR-SHP-003: FIFO例外
- ユーザーがFIFO順を変更する場合、理由入力は**不要**
